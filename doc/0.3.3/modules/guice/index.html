Guice Module

The Guice module provide a GuicePlugin that brings dependency injection thanks to Google Guice.

Tip
Guice Documentation

This documentation do not cover Guice itself as it is already well covered on the official user guide.

Declaration

In order to use the GuicePlugin you must add a dependency to org.qiweb.modules.guice and register the plugin in your configuration.

When using Gradle:

build.gradle 
compile 'org.qiweb:org.qiweb.modules.guice:{qiweb-version}'

When using Maven:

pom.xml 
<dependency>
    <groupId>org.qiweb</groupId>
    <artifactId>org.qiweb.modules.guice</artifactId>
    <version>{qiweb-version}</version>
</dependency>

And finally your application configuration:

application.conf 
app.plugins = [
    { plugin = "org.qiweb.modules.guice.GuicePlugin" }
]
